Search Engine Optimization (SEO) 101


Jargon

  • Search Engine: a program that returns search results based on keywords entered by users; some examples are Google, Yahoo, Bing and DuckDuckGo.
  • Keyword: a word or phrase entered into a search engine.
  • Keyword Stuffing: filling a website with keywords; penalized by Google.
  • Keyword Competitive Index (KPI): number of monthly searches for a keyword compared to the demand of the keyword.
  • Search Engine Optimization (SEO): methods to improve organic rank in search engine results; SEO is sometimes used as a job title.
  • Search Engine Marketing (SEM): SEO plus paid search advertising.
  • Search Engine Results Page (SERP): search engine page showing both organic and paid results.
  • Meta Description: webpage description that appears in search results.
  • Page Title: the web page name that shows up in the search results.
  • Alt Text: description added to a graphic or image.
  • Bot, spider, crawler: a program that scans a website's content.
  • Click-Through Rate (CTR): number of users who click on link compared to total number of users who see it.
  • Rank: how close to the top of the page a search result is.
  • Organic: unpaid search result.
  • Pay-Per-Click (PPC): search engine ads.
  • User Experience (UX): how easy or pleasing a website is to use.
  • Back or External Links: links on other sites that direct to your site.
  • Quality: level of content, UX, backlinks
  • Relevance: keywords, topics and location.
  • SEO or Link Juice: colloquial term sometimes used to donate practices that boost SEO rank; the more juice the better.
  • Index: home page of website.
  • Redirect: a method for sending users from removed content to new content.
  • A/B Test: sending slightly different web pages to different users to see which is most impactful.

Stats

  • 5 billion searches performed every day.
  • Google occupies almost 70% of the global search engine pie.
    Baidu is second with 18% percent.
    Yahoo and Bing combine for a little over 10%.
  • The top 3 search results get 63% of all clicks.
    The bottom 3 results (#7-10) get only 9% in total.
    Only 10-15% of clicks go to Adwords links (Google's PPC).

Reminders

  • There are no magic bullets.
  • SEO is more art than science.
  • Measure before and after any changes.

On-Page SEO

  • Keywords need to be present, but if keyword density is high, your site will be penalized for keyword stuffing.
